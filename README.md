# Alquiler Calatayud Website

This is the official website for Alquiler Calatayud, a premier accommodation service for workers, companies, and tourists in Calatayud, Spain.

## Features
- **Editorial Aesthetic**: High-end design based on the "Summit Peak Lodge" style.
- **Worker Focused**: Descriptions and amenities tailored for industrial/business team accommodation.
- **Fast Inquiry**: Integrated calendar and quick-mailto consultation for every property.
- **Fully Responsive**: Optimized for desktop and mobile viewing.

## Folder Structure
- `/index.html`: Main landing page with hero background of Calatayud.
- `/empresas.html`: Dedicated section for corporate partnerships and invoicing.
- `/sancho-gil-apt.html`: Detail page for Apartamento Sancho y Gil.
- `/sancho-gil-duplex.html`: Detail page for Duplex Sancho y Gil.
- `/san-andres.html`: Detail page for Apartamento San Andrés.
- `/img/`: Contains assets including the hero panoramic image.

## Hosting on GitHub Pages
1. Create a new repository on GitHub.
2. Upload all files from the `alquiler-calatayud/` folder.
3. Go to **Settings > Pages**.
4. Select the `main` branch and `/ (root)` folder.
5. Click **Save**.
